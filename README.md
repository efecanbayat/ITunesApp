# ITunesApplication


## About this project

This project is a application about iTunes which you can search movie, music, ebook, podcast.

## Used Technologies

* MVVM Architecture
* Room
* Mockito
* Dagger Hilt
* Retrofit, okHttp
* Kotlin Coroutines
* LiveData
* Gson
* Lottie Animations
* RecyclerView(Pagination with ScrollListener)
* Fragments
* Navigation Component

## Project Structure

* Search Screen
* Detail Screen
* Favorite Screen
